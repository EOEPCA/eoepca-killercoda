# EOEPCA Tutorial

> This tutorial requires the sysbox virtualisation engine. The CSI-rclone plugin used by the Workspace building block requires mount propagation features that are only available with sysbox. See [running on localcoda](https://github.com/EOEPCA/eoepca-killercoda#running-on-localcoda) for more information.

This tutorial requires [localcoda](https://github.com/spinto/localcoda) to run, it cannot be executed directly on the Killercoda cloud service due to resource limitations.

## Getting Started

See the [main EOEPCA Tutorials README](../README.md) for setup instructions and how to run this tutorial locally.

## Quick Start
```bash
# From the eoepca-killercoda directory
./run.sh <scenario-path>
```
