We have learned in this tutorial how to deploy a sample EOEPCA Processing Building Block with an [OGC API - Processes](https://ogcapi.ogc.org/processes/) interface and a Toil HPC execution engine!

We also followed a request across the main layers: ZOO-Project in Kubernetes, Toil WES, HTCondor, the application container, and S3-compatible object storage.

You can now experiment further with the deployed software or move to another [EOEPCA tutorial](https://killercoda.com/eoepca/).
